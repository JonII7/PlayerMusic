# 🎵 PlayerMusic

**Play music inside Minecraft using URLs from YouTube and other media services.**

PlayerMusic is a Minecraft mod that introduces a fully immersive in-game music system, allowing players to listen to their favorite songs directly inside the game world with high-quality audio playback.

> ⚠️ This mod is focused **only on music and audio immersion**, not video playback.

---

## ✨ Features

### 📻 Radio Block

* Adds a **Radio Block** to the game.
* Play music directly from a placed block in the world.
* Optional visual display showing playback progress on top of the block.

---

### 🔊 Music Command

Use the built-in command to play music for yourself or other players:

/playmusic "URL" <PLAYER | @a> <VOLUME>

**Parameters:**

* "URL" → Direct link to the media (YouTube or supported services)
* PLAYER → Play music for a specific player
* @a → Play for all players on the server
* VOLUME → Controls playback volume

> 🎧 This command plays **audio only** (no video).

## 📦 Required Dependencies

To use PlayerMusic correctly, install the following:

* WaterMedia
* CreativeCore
* WaterMedia YouTube Plugin

These dependencies ensure proper audio streaming and compatibility with YouTube sources.
